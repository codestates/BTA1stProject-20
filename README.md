# BTA1stProject-20
## Introduction
팜 월렛(Palm Wallet) 서비스는 니모닉 및 지갑 생성, 잔고 조회, 송금을 가능하게 하는 Palm에 특화된 Chrome Extension 지갑입니다.

## Getting Started

### Install
```shell
# client npm 의존성 설치
cd client && npm install

# server npm 의존성 설치
cd ../server && npm install
```

### Run
```shell
# client local web server 실행
cd client && npm run start

# api server 실행
cd ../server && npm run start
```
